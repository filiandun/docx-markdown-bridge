# Docx Markdown Editor

Convert DOCX files to Markdown and edit them directly in VS Code.

## Features

- Open `.docx` files as Markdown
- Edit Markdown in your preferred editor
- Automatically sync changes back to `.docx` on save (via Pandoc)

## Requirements

- Pandoc must be installed and available in PATH

## Usage

1. Right-click a `.docx` file
2. Click **Open as Markdown**
3. Edit the generated `.md` file
4. Save → `.docx` updates automatically

## Notes

- Temporary files are created as `.docx.md`
